Application designed for performing physiotherapy or rehabilitation exercises using a webcam.
On one hand, the person responsible for the patient can create custom exercises and sessions, as well as consulting their patients' data.
On the other hand, the patient can perform these exercise sessions or consult their own results.
It has been developed in Windows 10 using Python 3.10.11.

To initialise and download all the necessary dependencies (see requirements.txt), both an Instalacion.bat and an Instalacion.sh file are available. Once you have downloaded everything you need, you can run the GUI.py file.
